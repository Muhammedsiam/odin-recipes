# Odin Recipes

This is my first project from The Odin Project curriculum. A basic recipe website built with HTML to practice the fundamentals.

## Description

A simple recipe website featuring three recipes with ingredients, instructions, and images. The site demonstrates basic HTML structure, lists, links, and images.

## Skills Demonstrated

- HTML boilerplate structure
- Working with headings, paragraphs, lists
- Semantic HTML (figure, figcaption, nav)
- CSS styling, gradients, hover effects and transitions
- Linking pages and organizing file structure
- Git workflow and GitHub Pages deployment

## Recipes Included

1. Creamy Pasta Bake with Cherry Tomatoes and Basil
2. Chicken Stir Fry
3. Strawberry Pretzel Salad
